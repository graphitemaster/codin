# codin

An Odin to C compiler written in C.

### Building

To build on Linux
```
$ make
```

> On Linux you have some additional build options provided by the Makefile. Check the documentation at the top of the `Makefile` for them.

To build on Windows
```
$ cl.exe unity.c
```
